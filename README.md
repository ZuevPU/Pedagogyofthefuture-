# Педагогика будущего

Мини-проект по итогам образовательного мероприятия  
**«Педагогика будущего: внедрение современных технологий в образовательный процесс»**  
Центр знаний «Машук» · 11–12 июля 2026

## Содержание

Адаптивный лендинг программы с материалами лекций:

- презентации
- конспекты
- диагностика / саморефлексия

## Структура

```
landing/
  index.html                 — главная страница
  lecture-1-notes.html       — конспект лекции 01
  lecture-1-diagnosis.html   — диагностика «Паутина»
  diagnosis-1.js             — логика паутины
  styles.css                 — стили
```

## Запуск

Откройте `landing/index.html` в браузере или поднимите локальный сервер:

```bash
cd landing
python -m http.server 8765
```

Затем перейдите на http://127.0.0.1:8765/

1. Будущее образования: как работает современная цифровая среда (Четверов А. В.)

3. Когнитивное восприятие информации в цифровой образовательной среде (Миронова В. В.)
4. Образование сегодня сквозь призму глобальных процессов (Зуев П. Ю.)
